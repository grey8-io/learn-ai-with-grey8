# Rubric — Your First Git Repository

| Criterion | Points | Description |
|---|---|---|
| Directory created | 20 | `my-project` directory exists after running the script |
| Git initialized | 20 | `.git` folder is present inside `my-project` |
| File created | 20 | `hello.txt` exists with the correct content |
| File staged and committed | 30 | At least one commit exists in the Git log |
| Clean working tree | 10 | `git status` reports no untracked or modified files |

**Total: 100 points**

## Passing threshold

- **80+ points**: Pass
- **Below 80**: Review the lesson content and try again — you've got this!
