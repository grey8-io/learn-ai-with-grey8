# Rubric — AI Fact Checker

| Criterion | Points | Description |
|---|---|---|
| Verified claims detected | 25 | Correctly identifies claims matching known facts |
| Contradicted claims detected | 25 | Identifies claims that conflict with known facts |
| Unverified claims handled | 20 | Returns unverified for claims with no matching facts |
| Case insensitivity | 10 | Works regardless of input capitalization |
| Result structure correct | 10 | Returns list of dicts with claim, status, reason |
| Code quality | 10 | Clean code with docstrings and type hints |

**Total: 100 points**

## Passing threshold

- **80+ points**: Pass
- **Below 80**: Review the lesson content and try again — you've got this!
