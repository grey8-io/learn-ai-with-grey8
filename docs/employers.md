# For Employers

> Hire verified AI developers. Skip the screening.

---

## The Problem

Hiring AI talent is expensive and unreliable:
- Resumes don't prove someone can build
- Bootcamp certificates don't mean competence
- Technical interviews are time-consuming and inconsistent
- You won't know if they can actually ship until they're on the job

---

## How Grey8 Solves It

Every Grey8-certified developer has:

1. **Completed a rigorous curriculum** -- 12 phases, 35 lessons, 15 real projects covering Python, APIs, LLMs, RAG, AI agents, and deployment
2. **Passed a proctored assessment** -- timed, locked-down, code-based exam graded by automated tests + AI review
3. **A verified credential** -- publicly verifiable certificate with skills breakdown and composite score
4. **A project portfolio** -- real projects built during the bootcamp, not toy examples

---

## What You Can Do

### Search Candidates

Browse certified candidates filtered by:
- **Skill** -- Python, FastAPI, RAG, agents, deployment, etc.
- **Score** -- minimum composite score threshold
- **Availability** -- ready now, available in 2 weeks, etc.

### Verify Credentials

Every Grey8 certificate has a verification code. Visit the public verification page to confirm:
- Certificate is genuine
- Candidate name and skills
- Composite score
- Issue date

Example: `enterprise.grey8.dev/verify/G8-A3F7K9`

### Request Placements

Found a candidate you like? Submit a placement request through the employer portal. Grey8 facilitates the introduction, handles contracting, and supports the placement.

### Run a Team Bootcamp

Need to upskill your existing team? Grey8 runs cohort-based bootcamps:
- Same battle-tested curriculum
- Custom assessment tracks for your team's needs
- Training manager dashboard with real-time progress
- Team members earn Grey8 certification upon completion

## Why Grey8?

- **Open training, rigorous assessment** -- the curriculum is public so you can see exactly what candidates learned. The assessment is proctored so you know they actually learned it.
- **Code-based proof** -- no multiple choice. Candidates write real code in timed, proctored conditions.
- **Transparent skills** -- every candidate has a skills matrix with scores per domain, not just a pass/fail badge.
- **Lower hiring risk** -- you see the candidate's assessment score, project portfolio, and verification before you commit.
