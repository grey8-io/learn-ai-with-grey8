> Prove your AI skills with a proctored, verifiable credential.

## What Is Grey8 Certification?

A Grey8 certificate proves that you can build real AI applications -- not just answer quiz questions. The assessment is proctored, code-based, and graded by both automated tests and AI review.

Every certificate comes with a **public verification URL** that anyone (including employers) can check instantly.

## What the Assessment Covers

The assessment draws from the **same exercises** you practiced in the bootcamp -- but you take them in a locked-down environment without hints, solutions, or the AI tutor.

| Skill Area | Examples |
|-----------|----------|
| Python fundamentals | Data structures, functions, file I/O |
| API development | FastAPI, REST, request handling |
| AI/LLM integration | Ollama API, prompt engineering, structured outputs |
| RAG systems | Embeddings, vector search, document QA |
| AI agents | Tool use, agent loops, multi-step reasoning |
| Deployment | Docker, async patterns, production best practices |

Each candidate receives a **randomly selected subset** of exercises from the bootcamp pool. You won't know which specific exercises will appear -- so you need to genuinely understand the material, not memorize solutions.

The difference is not the questions -- it's the conditions. If you truly learned the skills, you'll pass. If you relied on hints and solutions, the proctored environment will expose that.

## How Proctoring Works

You use the same code editor you practiced with in the bootcamp. The difference is what's disabled:

| Feature | During Learning | During Exam |
|---------|----------------|-------------|
| Solution reveal | Available after 5 attempts | Disabled |
| Hints (3 levels) | Available | Disabled |
| AI Tutor chat | Available | Disabled |
| Time limit | None | Fixed (auto-submit on expiry) |
| Tab switching | Allowed | Logged |

Additional rules:
- **One attempt** -- you can retake after a cooldown period, but each attempt is a full fresh assessment with a new random selection of exercises.
- **No external help** -- tab switches and focus changes are logged as proctor events.

This is not surveillance. There's no webcam or screen recording. The proctor log simply records whether you stayed in the exam environment.

## Your Certificate

When you pass, you receive:

- **Digital certificate** with your name, skills, and composite score
- **Verification code** (e.g., `G8-A3F7K9`) -- a short, unique code tied to your certificate
- **Public verification page** -- anyone can visit `enterprise.grey8.dev/verify/G8-A3F7K9` to confirm your credential is real
- **Candidate profile** -- your skills matrix and portfolio become visible to employers on the Grey8 platform

## FAQ

**How much does certification cost?**
$200-500 depending on the assessment level. Pricing details will be published when the enterprise portal launches.

**Can I retake the assessment?**
Yes, after a cooldown period. Each attempt is independently proctored and graded.

**Do I need to complete the entire bootcamp first?**
The assessment covers bootcamp material. You can attempt it anytime, but you're unlikely to pass without completing the curriculum.

**Is the certificate recognized by employers?**
Grey8 certification is new. Early recognition comes from direct placements -- when an employer hires a Grey8-certified developer and they perform well, the credential builds trust. We're building that track record now.

**What happens after certification?**
You become eligible for Grey8 placement. Your profile is visible to employers, and Grey8 may reach out for interview and placement opportunities.
